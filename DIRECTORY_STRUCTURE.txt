Key Files Explained
===================

Backend Files:
--------------
main.py                    - FastAPI application setup, CORS, routes
requirements.txt           - Python package dependencies
.env                       - API keys and configuration (YOU CREATE THIS)

app/api/routes.py          - API endpoints (/generate, /troubleshoot, etc.)
app/core/config.py         - Configuration management with Pydantic
app/models/protocol.py     - Request/response models, enums
app/prompts/*.py           - Prompt templates for LLM
app/services/llm_service.py - LLM provider integration (OpenAI, Anthropic)
app/services/protocol_service.py - Business logic for protocols

Frontend Files:
---------------
package.json               - Node.js dependencies and scripts
vite.config.js             - Vite build configuration
tailwind.config.js         - TailwindCSS theme configuration
index.html                 - HTML template

src/main.jsx               - React app initialization
src/App.jsx                - Main app with routing
src/index.css              - Global styles and TailwindCSS

src/components/Layout.jsx  - Header, navigation, footer
src/components/ProtocolDisplay.jsx - Markdown rendering, copy/download

src/pages/Home.jsx         - Landing page with features
src/pages/GenerateProtocol.jsx - Protocol generation form
src/pages/TroubleshootProtocol.jsx - Troubleshooting form

src/services/api.js        - Axios-based API client
src/utils/cn.js            - TailwindCSS class merging utility

Important Notes
===============
1. Create backend/.env from backend/.env.example
2. Add your API keys to backend/.env
3. Run backend and frontend in separate terminals
4. Backend runs on port 8000
5. Frontend runs on port 5173
6. API docs available at http://localhost:8000/docs


Quick Commands
==============
Backend:
  cd backend
  python -m venv venv
  venv\Scripts\activate
  pip install -r requirements.txt
  python main.py

Frontend:
  cd frontend
  npm install
  npm run dev

Or use the batch scripts:
  start-backend.bat
  start-frontend.bat
